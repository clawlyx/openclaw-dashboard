# Requirements Archive: v1.5.0 Coordination Clarity

**Archived:** 2026-03-21
**Status:** ✅ SHIPPED
**Outcome:** 9/9 milestone requirements validated, 0 adjusted, 0 dropped

## Milestone Goal

Make coordination across `Agents` and `Mission Control` easier to trust and faster to act on by clarifying task-to-agent mapping, overlap and handoff state, and the operator's next move without introducing new control surfaces.

## Requirement Outcomes

### Validated Requirements

- [x] **MAP-01**: Operator can see the current Mission Control task linked to a working agent when a live mapping exists.
  Outcome: working agents now expose linked Mission Control task context directly from the shared snapshot contract.
- [x] **MAP-02**: Operator can tell whether a shown task-to-agent mapping is exact, partial, or unavailable without mistaking it for ownership reassignment.
  Outcome: mapping confidence is now explicit in Agents and remains visibly advisory relative to Mission Control task truth.
- [x] **MAP-03**: Operator can open the relevant Mission Control context from the Agents coordination surface without manually searching for the task.
  Outcome: exact and partial mappings now hand off into the closest Mission Control task or review context with carried focus hints.
- [x] **HAND-01**: Operator can see when multiple agents are serving the same mission or closely related work so intentional parallelism and accidental overlap are distinguishable.
  Outcome: overlap groups now separate healthy shared work from ambiguous overlap using the shared coordination model.
- [x] **HAND-02**: Operator can see recent handoff state for active work, including the last agent involved and the expected next owner or lane when that data exists.
  Outcome: recent handoff summaries now surface the last agent plus next owner, lane, or queue when the evidence supports it.
- [x] **HAND-03**: Operator can tell when overlap or handoff ambiguity needs intervention more urgently than routine parallel work.
  Outcome: routine, watch, and intervene priority now appears in both Agents and Mission Control without creating a new workspace.
- [x] **NEXT-01**: Operator can see one clear recommended next action in the default Agents coordination surface for the current highest-priority coordination issue.
  Outcome: the default Agents scan path now highlights one recommended move from the current coordination evidence.
- [x] **NEXT-02**: Operator can tell whether the recommended next action belongs in Agents or Mission Control before leaving the default scan path.
  Outcome: recommendations now label the destination surface explicitly before the operator navigates.
- [x] **NEXT-03**: Operator can see concise reasoning for the recommended next action so the recommendation reads as trustworthy guidance instead of a guess.
  Outcome: recommendation reasons are now short, evidence-based, and mirrored on Mission Control landings when applicable.

### Deferred Requirements

- **FORE-01**: Operator can see likely coordination bottlenecks before pressure escalates.
- **FORE-02**: Operator can see trend-based handoff risk across rooms or missions.
- **CTRL-01**: Operator can start, stop, or restart host services and worker processes from the workstation.
- **CTRL-02**: Operator can see command outcomes and recovery state for host-control actions.
- **SURF-01**: Operator can open deeper coordination analytics in a dedicated surface when the current `Agents` and `Mission Control` views are no longer sufficient.

### Out of Scope at Archive Time

| Feature | Reason |
|---------|--------|
| Predictive bottleneck forecasting | Explicitly deferred by the `v1.5.0 Coordination Clarity` brief |
| Host service and worker-process controls | Explicitly deferred by the `v1.5.0 Coordination Clarity` brief |
| Dedicated net-new coordination workspace | Prioritize clarity on known `Agents` and `Mission Control` surfaces first |
| Task ownership truth moving into Agents | Mission Control remains the ownership system of record |
| Automatic assignment or autonomous staffing | Guidance must stay advisory and trustworthy |
